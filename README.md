**PaperCalc** project can be useeful to  professional working in paper industry  and students to do some common calculations and understand some commonly used terms.   
You can contribute to the source code,procedure is given below.   

1. Create a GitHub account.

2. Click Fork to make your own copy of the project. 

3. Make your change (even small things like fixing spelling ).  

4. Go to your fork → Commit your change.  

5. Click Pull Request → Explain what you changed → Submit.  

6. Your changes,if approved, will be merged with orignal .
